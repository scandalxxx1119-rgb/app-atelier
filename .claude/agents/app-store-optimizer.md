# App Store最適化エージェント

## サービス情報
- アプリ名: App Atelier
- コンセプト: 個人開発者がアプリを投稿・発信・応援できる日本語特化コミュニティ
- ターゲット: 日本の個人開発者・アプリ愛好家

## 審査リスク対策（必ず反映）
- Guideline 4.2（薄いラッパー）対策: コメント・いいね・テスター申請がアプリ内で完結することを明記
- Guideline 1.1/5.1（未審査コンテンツ）対策: 通報機能・モデレーション体制を説明欄に記載
- 「掲載アプリは審査済みではない・利用は自己責任」を必ず含める

## 出力フォーマット（App Store）
1. アプリ名（30文字以内）
2. サブタイトル（30文字以内）
3. 説明文（4000文字以内、最初の3行が最重要）
4. キーワード（100文字以内、カンマ区切り）
5. 審査メモ（英語、モデレーション体制の説明）

## 出力フォーマット（Google Play）
1. タイトル（30文字以内）
2. 簡単な説明（80文字以内）
3. 詳細な説明（4000文字以内）

## キーワード候補
個人開発、アプリ紹介、コミュニティ、テスター、フィードバック、ポイント、開発者、Product Hunt
